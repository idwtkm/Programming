# HumanBeing Collection Manager 👬

Консольное приложение для управления коллекцией 'HumanBeing'

## Особенности 🤠
- **LinkedHashMap**: Для создания коллекции используется определенный тип LinkedHashMap
- **Запись из файла**:При запуске приложения коллекция должна автоматически заполняться значениями из файла
- **XML**: Данные в файле должны храниться в формате xml
- **java.io.PrintWriter** для ввода
- **java.io.BufferedInputStream** для вывода
- **JAVADOC**: Все классы в программе должны быть задокументированы в формате javadoc.

## Команды приложения 📋
- **help** : вывести справку по доступным командам
- **info** : вывести в стандартный поток вывода информацию о коллекции (тип, дата инициализации, количество элементов и т.д.)
- **show** : вывести в стандартный поток вывода все элементы коллекции в строковом представлении
- **insert null {element}**: добавить новый элемент с заданным ключом
- **update id {element}** : обновить значение элемента коллекции, id которого равен заданному
- **remove_key null** : удалить элемент из коллекции по его ключу
- **clear** : очистить коллекцию
- **save** : сохранить коллекцию в файл
- **execute_script file_name**: считать и исполнить скрипт из указанного файла. В скрипте содержатся команды в таком же виде, в котором их вводит пользователь - в интерактивном режиме.
- **exit** : завершить программу (без сохранения в файл)
- **remove_greater {element}** : удалить из коллекции все элементы, превышающие заданный
- **replace_if_lowe null {element}** : заменить значение по ключу, если новое значение меньше старого
- **remove_lower_key null**: удалить из коллекции все элементы, ключ которых меньше, чем заданный
- **remove_any_by_car car**: удалить из коллекции один элемент, значение поля car которого эквивалентно заданному
- **average_of_minutes_of_waiting**: вывести среднее значение поля minutesOfWaiting для всех элементов коллекции
- **print_ascending**: вывести элементы коллекции в порядке возрастания

## Настройка и управление 🔧
Имя файла должно передаваться программе с помощью переменоной окружения
